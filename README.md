# Marketplace Analytics

Финальный проект курса Data Analytics. Аналитика онлайн-маркетплейса
формата Ozon: ежедневный сбор данных по API в PostgreSQL, дашборд в Metabase
и два аналитических исследования за 2023 год (ABC×XYZ, RFM + когорты).

## Стек

Python (requests, psycopg2, pandas, loguru), PostgreSQL 16, Docker, cron, Metabase.
